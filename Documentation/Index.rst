PDF Lister
=============================================================

.. only:: html

	:Classification:
		pdf_lister

	:Version:
		|release|

	:Language:
		en

	:Description:

		This extension lists PDF files from a selected Folder.

		The website users can download these PDF files securely, without knowing the actual PDF path.

		The extension also allows signing up such downloaded PDF files with a custom message.
